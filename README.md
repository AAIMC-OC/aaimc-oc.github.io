
# AAIMC‑OC Consortium  
## Democratising AI diagnostics for equitable cancer care

**AAIMC‑OC is a global platform aiming to develop and evaluate AI‑defined cancer diagnostics using federated learning, enabling patients worldwide to benefit from advances in precision medicine.**

This is the official webpage of the Application of AI-defined Molecular Classifications in Ovarian Cancer (AAIMC-OC) Consortium
